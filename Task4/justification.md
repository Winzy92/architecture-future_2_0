## Виртуальные машины

- Data Platform: 4 CPU / 8 GB — для обработки данных
- Kafka: 2 CPU / 4 GB — потоковые данные
- BI: 2 CPU / 4 GB — аналитика

## Сеть

- VPC + Subnet обеспечивают изоляцию
- NAT позволяет безопасный доступ к интернету
- Security Groups ограничивают доступ

## IaC (Terraform)

- инфраструктура воспроизводима
- можно быстро развернуть новые окружения
- снижает риск ошибок

## Связь с бизнесом

- масштабируемость → рост компании
- безопасность → защита чувствительных данных
- гибкость → быстрый запуск и интеграция новых сервисов
